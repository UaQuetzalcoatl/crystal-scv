# crystal-app

Symple formatting app

## Installation


* Install Crystal
* Install dependencies
```sh
shards install
```
* Build app
```sh
crystal build src/index.cr
```


## Usage

* start the app
```sh
./index
```

* try to make request `http://localhost:3000`
